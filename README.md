# maru-hi

Synthesis 2 image.

## usage
Synthesis 2 image.
``` 
cd ~/golang/src/github.com/athagi/maru-hi
go build
./maru-hi -p dst.jpeg -l layer.png
```

or 

Write a text on base image.
```
cd ~/golang/src/github.com/athagi/maru-hi
go build
./maru-hi -p dst.jpeg -t "this is sample"
```
